javascript中的数据类型

###基本数据类型

  1. string
  2. number
  3. boolean

###引用数据类型

  1. object
  2. function
  3. Array
  4. RegExp
  5. Date
  6. Math
  7. Boolean
  8. Number
  9. String

要点tips

  1. 对象类型的数据进行对比,是比较的存储地址
  2. 当一个值类型的数据和一个引用类型的数据发生运算时,遵循如下规则:
    先调用这个引用类型数据的valueOf方法,获取返回值之后参与运算,如果能运算,那么直接得出结果
    如果不能运算,继续调用该引用类型数据的toString方法,获取返回值参与运算,这个是可以出来结果的.
    引用类型除了Date之外用valueOf方法返回的是自身
    {}.toString() 返回的是[object Object]
    [].toString() 返回的是’’ 空字符串

  2. []关联数组语法 对象名[属性名] =值

值类型和引用类型

  1. 值类型:
    直接存储的就是数据本身的这种数据,就是值类型的数据
  2. 引用类型
    存储的是数据的地址,数据单独存起来的这种数据,就是引用类型的数据

特征

  • 当值类型的数据进行赋值的时候,会将现有的变量存储的数据直接复制一份交给新的变量,此时两个变量虽然纸一样,但是存储的空间完全独立,修改其中一个,另一个不受影响

  • 引用类型的数据在赋值的时候,会将变量中存储的对象的地址复制一份交给新的变量,此时两个变量同时指向了同一个对象,修改其中一个,另外一个也会受影响

  • 简单来说 值类型赋值的数据,引用类型赋值的是数据的存储地址

面向对象的基本概念

  • 面向过程
  • 是一种解决问题的思维方式,关注点在解决问题的步骤上
  • 面向对象

  • 是一种解决问题的思维方式,关注点在就解决问题所需要的一系列对象上

    面向对象就是对面向过程的封装
