构造函数的相关概念

构造函数是什么

构造函数就是一个函数,构造函数一般被用来初始化对象

构造函数的特点

  1. 一般情况下,首字母大写
  2. 一般情况下,和new 关键字一起使用
  3. 构造函数不需要自己手动去写return语句,会自动返回创建出来的对象

构造函数的执行过程

  1. new关键字创建对象
  2. 调用构造函数,将构造函数内部的this指向new创建出来的对象
  3. 执行构造函数的代码,通过this为刚才创建出来的对象新增属性和方法
  4. 默认自动返回了刚才创建出来并且已经田家湾属性和方法的对象

构造函数的注意事项

  1. 如果手动写了return语句
    1.1 如果return的是值类型的数据,name不会对默认返回值产生影响
    1.2 如果return的是引用类型的数据,那么返回的是就是return语句后面的数据
  2. 如果把构造函数当做普通函数来调用
    2.1 函数中的this会指向window,通过this添加的所有的属性和方法全部添加给了window
    2.2 此时构造函数的返回值,完全按照普通函数来理解

传统的构造函数写法存在的问题

  1. 如果将方法的声明放在构造函数里面,name每次创建对象方法声明都会被执行一次,生成一个新的方法
  2. 每个对象都拥有一个独立的这个方法,但是,所有对象拥有的方法功能都是完全一致的,这样就造成了资源浪费

原型的基本概念

原型是什么

当构造函数被创建出来的时候,系统会默认的办这个构造函数创建并且关联一个空的对象,这个空的对象就是原型

原型的作用

原型中所有的属性和方法,可以被和其关联的构造函数所创建出来的所有的对象共享

如何访问原型
